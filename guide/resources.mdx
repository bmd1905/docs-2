# Resources

Synchronize your cloud resources with CloudThinker so your AI agents can monitor, analyze, and optimize your infrastructure.

<Card title="Configure Resource Sync" icon="sync" href="#configuring-sync">
  Set up automatic synchronization with your cloud resources
</Card>

---

## Configuring Sync

<Steps>
  <Step title="Open Sync Configuration">
    Navigate to Resources and click sync settings
  </Step>
  <Step title="Select Services">
    Choose which AWS services to synchronize
  </Step>
  <Step title="Set Sync Options">
    Configure refresh interval and include stopped resources
  </Step>
  <Step title="Save Configuration">
    Apply settings to start synchronization
  </Step>
</Steps>

## Sync Options

<CardGroup cols={2}>
  <Card title="Include Stopped Resources" icon="toggle-on">
    Enable to sync resources that are currently stopped or inactive for complete visibility
  </Card>
  <Card title="Auto-refresh Interval" icon="clock">
    Set how often to sync resources (recommended: 30 minutes)
  </Card>
</CardGroup>

## Getting Started

<Steps>
  <Step title="Start with Core Services">
    Select EC2, RDS, S3, and VPC for essential monitoring
  </Step>
  <Step title="Configure Sync Options">
    Enable stopped resources and set 30-minute refresh interval
  </Step>
  <Step title="Save Configuration">
    Apply settings to start synchronizing your resources
  </Step>
</Steps>
