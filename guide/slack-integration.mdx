## Installation & Setup

### Step 1: Install CloudThinker Slack App

<Steps>
  <Step title="Access Slack Integration">
    Navigate to **Settings > Integrations > Slack** in the CloudThinker console
  </Step>
  <Step title="Install to Slack">
    Click **"Add to Slack"** and authorize the CloudThinker app with required permissions:
    - Read channel information
    - Send messages and attachments
    - Access user information
    - Manage app mentions and direct messages
  </Step>
  <Step title="Select Workspace">
    Choose the Slack workspace where you want to install CloudThinker
  </Step>
</Steps>

## Troubleshooting

### Common Issues & Solutions

<Accordion title="üîå Connection Issues">
  **Symptoms:** Agents not responding to mentions
  
  **Solutions:**
  1. Check CloudThinker app permissions in Slack
  2. Verify agent is configured for the channel
  3. Ensure CloudThinker service is running
  4. Re-authenticate Slack integration if needed
  
  **Diagnostic Commands:**
  ```
  /cloudthinker status
  /cloudthinker test-connection
  ```
</Accordion>

<Accordion title="üîî Missing Alerts">
  **Symptoms:** Not receiving expected alerts
  
  **Solutions:**
  1. Verify alert configuration in CloudThinker console
  2. Check Slack channel notification settings
  3. Confirm alert thresholds are properly set
  4. Review alert history for debugging
  
  **Configuration Check:**
  ```
  /cloudthinker alerts config
  /cloudthinker alerts test
  ```
</Accordion>

<Accordion title="üö´ Permission Errors">
  **Symptoms:** "Access denied" or permission-related errors
  
  **Solutions:**
  1. Review Slack app permissions
  2. Check CloudThinker workspace access
  3. Verify agent channel permissions
  4. Re-install CloudThinker app if necessary
  
  **Permission Audit:**
  ```
  /cloudthinker permissions check
  /cloudthinker permissions repair
  ```
</Accordion>

## Best Practices

### Channel Organization

<Steps>
  <Step title="Separate by Function">
    Create dedicated channels for different operational areas (costs, security, database, etc.)
  </Step>
  <Step title="Use Clear Naming">
    Follow consistent naming conventions like `#cloudthinker-[function]`
  </Step>
  <Step title="Set Channel Purposes">
    Document each channel's purpose and which agents have access
  </Step>
  <Step title="Configure Notifications">
    Set appropriate notification levels to avoid alert fatigue
  </Step>
</Steps>
