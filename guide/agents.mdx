## Configuration

<Steps>
  <Step title="Select Tools">
    Choose from chart, dashboard, report, alert, and search tools
  </Step>
  <Step title="Assign Connections">
    Connect agents to your cloud resources and databases
  </Step>
  <Step title="Set Instructions">
    Define custom behavior and operational guidelines
  </Step>
</Steps>

<Note>
Tool approval is configured in [Connections](/guide/connections) under Built-in Tools.
</Note>

---

## Connections

Assign connections based on agent expertise:
- **Database agents**: Database connections
- **Security agents**: Security monitoring connections
- **Cloud engineers**: Cloud provider connections
- **K8s agents**: Kubernetes cluster access

<Note>
Connection setup is managed in [Connections](/guide/connections).
</Note>

---

## Instructions

Define agent behavior through custom instructions:
- **Role & Expertise**: Specify the agent's background and capabilities
- **Scope & Permissions**: Set access boundaries and limitations
- **Operational Guidelines**: Define procedures and escalation paths
