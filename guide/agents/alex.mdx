## Common Use Cases

### Cost Analysis
**Example:** `@alex analyze EC2 instances with <20% CPU utilization over past 30 days`

**What you get:** Right-sizing recommendations with potential cost savings and migration steps.

### Performance Review
**Example:** `@alex review our Auto Scaling groups and recommend scaling optimizations`

**What you get:** Scaling policy improvements with performance impact analysis.

### Multi-Cloud Comparison
**Example:** `@alex compare AWS vs Azure costs for our current workloads`

**What you get:** Cost comparison matrix with migration complexity assessment.

### Infrastructure Audit
**Example:** `@alex audit our cloud infrastructure for optimization opportunities`

**What you get:** Comprehensive report with prioritized recommendations.

## Best Practices

**Be specific with your requests:**
```
✅ @alex analyze EC2 instances in us-east-1 with <15% CPU utilization over past 14 days
❌ @alex check our servers
```

**Include context:**
```
✅ @alex review production RDS instances for cost optimization while maintaining 99.9% uptime
❌ @alex optimize our databases
```

**Specify timeframes:**
```
✅ @alex analyze cost trends over the last 6 months
❌ @alex look at our costs
```

## Configuration

**Supported Platforms:** AWS, Azure, GCP
**Required Tools:** `#recommend`, `#chart`, `#report`, `#dashboard`, `#alert`
**Connections:** Cloud accounts with read access to cost and performance metrics

## Getting Started

1. Connect your cloud accounts
2. Start with: `@alex analyze our current cloud spending`
3. Review recommendations and implement changes
4. Set up monitoring dashboards for ongoing optimization
