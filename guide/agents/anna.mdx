## Common Use Cases

### Multi-Cloud Migration
**Example:** `@anna orchestrate AWS to Azure migration project coordinating @alex for infrastructure, @oliver for security, @tony for database migration, and @kai for containerization`

**What you get:** Complete migration plan with agent-specific tasks, dependencies, timeline, and risk mitigation strategies.

### Complex Infrastructure Project
**Example:** `@anna manage enterprise-wide Kubernetes adoption project involving security hardening, database optimization, and cost management across all cloud platforms`

**What you get:** Multi-phase project plan with @kai handling K8s deployment, @oliver managing security policies, @alex optimizing costs, and @tony handling data layer.

### Cloud Security Transformation
**Example:** `@anna coordinate comprehensive security audit and remediation across multi-cloud infrastructure with compliance requirements`

**What you get:** End-to-end security transformation with @oliver leading security assessment, @alex handling infrastructure changes, @kai securing containers, @tony protecting databases.

### Enterprise Cloud Optimization
**Example:** `@anna optimize our entire cloud infrastructure for cost, performance, and security across AWS, Azure, and GCP environments`

**What you get:** Holistic optimization strategy with each agent handling their specialty: @alex (cost optimization), @oliver (security), @tony (database performance), @kai (container efficiency).
